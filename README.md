#  🎶 Projeto - DexPlayer 🎶
- Este é um player de música simples, mas eficaz, o usuário pode adicionar suas musicas na pasta de musicas e o projeto irá mostralas no terminal, com as seguintes funções
  
#  🎶 Funções 🎶
-  ⏮️ Voltar música
-  ▶️Começar música
-  ⏹️Parar música
-  ⏭️Passar música
-  📖Listar músicas
